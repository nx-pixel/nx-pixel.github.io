📖 О симуляции

Это эволюционная модель «Хищник-Жертва». Жертвы (жёлтые) ищут еду (зелёные точки) и размножаются. Хищники (красные) охотятся на жертв.

При размножении есть шанс мутации — параметры потомка меняются, и он получает новый символ в эволюционной цепи.

🖱️ Управление

  • ЛКМ по карте — добавить хищника
  • ПКМ по карте — добавить жертву
  • ⏸ Пауза / ▶ Старт — остановить/запустить симуляцию
  • ⟳ Сброс — перезапустить симуляцию
  • 📊 График — открыть график популяций
  • 📋 Виды — открыть таблицу всех видов
  • ? — открыть эту справку

⚙️ Параметры

  • Скорость симуляции — ускорение/замедление времени
  • Вероятность мутации — шанс мутации при размножении
  • Сила мутаций — насколько сильно меняются параметры
  • Мобильность — скорость перемещения
  • Осторожность — вероятность убегать от хищника
  • Автопауза — автоматическая пауза при вымирании

🧬 Эволюционные цепи

Жертвы: 🐇→🐑→🐐→🐄→🐏→🦌→🦒→🐘
Хищники: 🐊→🐍→🦅→🦉→🐆→🐺→🦁→🐯

При мутации существо получает следующий символ в цепи. При достижении конца цепи — возвращается к первому символу.

📊 Виды

Каждый уникальный вид (название + номер мутации) регистрируется в базе видов. В таблице можно увидеть историю каждого вида: время жизни, пик численности, параметры и потомков.
